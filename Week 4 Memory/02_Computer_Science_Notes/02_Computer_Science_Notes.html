ملاحظات علوم الحاسب: جبر المؤشرات ومحاذاة الذاكرة

جبر المؤشرات (Pointer Arithmetic)

عندما نجمع رقماً مع مؤشر، فإن المعالج لا يجمع بايتات فردية، بل يضرب الرقم في حجم نوع البيانات (sizeof). هذا يضمن القفز المباشر بين عناصر المصفوفة. 

===================================================================================
             POINTER ARITHMETIC MULTIPLICATION MATRIX
===================================================================================

  [ Base Address: 0x1000 ] (int *p)
   │
   ├─► p + 1 ──> 0x1000 + (1 * 4 bytes) ──> 0x1004
   ├─► p + 2 ──> 0x1000 + (2 * 4 bytes) ──> 0x1008
   └─► p + 3 ──> 0x1000 + (3 * 4 bytes) ──> 0x100C

===================================================================================

محاذاة الذاكرة التلقائية (Memory Alignment)

تقرأ المعالجات الحديثة الذاكرة في كتل بحجم 8 بايت. يقوم المترجم بحقن بايتات فارغة (Padding Bytes) داخل الهياكل (structs) لضمان المحاذاة التلقائية وتسريع دورة الجلب.

التصادم الحتمي: طفح المكدس مقابل طفح البافر

يحدث طفح المكدس (Stack Overflow) بسبب الاستدعاءات الذاتية اللانهائية، بينما يحدث طفح البافر (Buffer Overflow) عند كتابة بيانات تتجاوز مساحة المصفوفة المخصصة، مما يفتح ثغرات أمنية خطيرة.

Computer Science Notes: Pointer Algebra & Alignment

Pointer Arithmetic Multipliers

Adding an integer to a pointer instructs the underlying ALU to advance the base address by K * sizeof(Type) bytes, ensuring flawless traversal across contiguous physical storage arrays. 

===================================================================================
             POINTER ARITHMETIC MULTIPLICATION MATRIX
===================================================================================

  [ Base Address: 0x1000 ] (int *p)
   │
   ├─► p + 1 ──> 0x1000 + (1 * 4 bytes) ──> 0x1004
   ├─► p + 2 ──> 0x1000 + (2 * 4 bytes) ──> 0x1008
   └─► p + 3 ──> 0x1000 + (3 * 4 bytes) ──> 0x100C

===================================================================================

Struct Padding & Data Alignment

Modern microprocessors fetch memory in uniform 8-byte chunks. The compiler injects uninitialized Padding Bytes to ensure natural word alignment and prevent hardware clock cycle penalties.

Asymptotic Boundaries: Stack vs. Buffer Overflow

Stack Overflow is triggered by unbounded recursion exhausting stack frames, whereas Buffer Overflow corrupts adjacent memory buffers and overwrites base function return address pointers.
