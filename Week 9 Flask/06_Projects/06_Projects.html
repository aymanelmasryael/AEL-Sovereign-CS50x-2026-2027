تطبيق AEL Flask MVC Core

تطبيقاً لمبدأ السيادة البرمجية الكاملة وتجنب النسخ الحرفي لمشاريع CS50 التقليدية، تم بناء منظومة ويب متكاملة (Full-Stack Web Suite) مبنية على إطار Flask مع تطبيق هيكلي صارم لنمط التصميم MVC داخل خزانة ael_src9.

الوحدات المعمارية المطبقة

  • محرك بكرة التوجيه (Master Controller Engine): نصوص Python منظمة تدير استعلامات GET / POST مع طبقات فحص أمني وتطهير للبيانات.
  • حقن قوالب الجيل الثالث (Jinja2 Glassmorphism Core): قوالب HTML معززة بشروط وحلقات Jinja، مُصممة وفق أرقى المعايير الجمالية للوضع الداكن الزجاجي.
  • نظام التتبع الجلسي (Cryptographic Session Inspector): دوال لمراقبة توليد الرموز المشفرة وتفكيك الجلسات عند تسجيل الخروج.
  • محاكي طبقة قاعدة البيانات (SQLite Abstracted Transactions): عمليات دمج تحاكي إنشاء وقراءة وتحديث السجلات بصورة مستمرة.
