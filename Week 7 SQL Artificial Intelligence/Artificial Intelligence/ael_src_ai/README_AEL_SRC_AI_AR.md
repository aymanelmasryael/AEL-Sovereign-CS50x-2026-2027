## 🚀 دليل تشغيل الشفرات السيادية للذكاء الاصطناعي (`ael_src_ai`)

تم بناء ترسانة برمجية كاملة وحصرية 100% داخل `ael_src_ai`، خالية تماماً من أي شفرات قديمة أو منقولة، لتغطي النماذج اللغوية الكبيرة (`LLMs`)، التحليل الدلالي، ومحركات أشجار القرار.

### 1. موصل النماذج اللغوية السيادي (`ael_llm_connector.py`)
```bash
python3 ael_llm_connector.py
```
*   **المهمة المعمارية**: محاكاة اتصال مشفر مع واجهات النماذج اللغوية الكبيرة (`LLMs - GPT-5/GPT-6`)، وتطبيق طبقة صارمة من تعليمات النظام (`System Prompt`) لمنع هجمات حقن التلقين (`Prompt Injection`).

### 2. محرك فحص الكلمات والتنبؤ الدلالي (`ael_ai_speller_matrix.py`)
```bash
python3 ael_ai_speller_matrix.py
```
*   **المهمة المعمارية**: دمج مفاهيم القاموس التقليدي (`Hash Tables / Tries`) مع خوارزميات التصحيح الذكية القائمة على مسافة ليفنشتاين (`Levenshtein Distance`) والتنبؤ الاحتمالي بالأخطاء.

### 3. محرك أشجار القرار والتقليم الألفا-بيتا (`ael_decision_tree_engine.py`)
```bash
python3 ael_decision_tree_engine.py
```
*   **المهمة المعمارية**: محاكاة شجرة قرارات ذكية باستخدام خوارزمية `Minimax` وتقنية التقليم `Alpha-Beta Pruning` لحساب المسار التكتيكي الأفضل استهلاكاً للموارد.
